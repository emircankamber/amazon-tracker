Google Sheets Senkronizasyonu

Her fiyat kontrolünden sonra veriler otomatik olarak Google Sheets'e yazılır.

Aktifleştirmek için Railway'de şu iki variable'ı ekle:

GOOGLE_SHEET_ID — Sheet URL'sindeki ID (docs.google.com/spreadsheets/d/BU_KISIM/edit)

GOOGLE_SERVICE_JSON — Google Cloud Service Account JSON içeriği (tek satır)

Service Account için: Google Cloud Console → IAM → Service Accounts → JSON key indir → Sheet'i bu hesapla paylaş (Editor)
